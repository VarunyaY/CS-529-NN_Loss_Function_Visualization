How to use?
  • Use mouse to rotate the axes.
  • Use arrow keys to move the origin.
  • WireFrame and auto rotate options can be enabled to get the wireframe representation and rotate the scene.
  • Reset button resets the camera angle.
  • Use the slider to view the X, Y filter normalized directions for a particular loss value.

Disclaimer
  • The Visualization only focusses on four models, namely: Resnet, Densenet, Renset no Short and VGG.
  • The Red, Green and Blue lines indicate the directions of positive X, Y and Z axes.
  • Please select a model to view and interact with slider to know the X, Y direction values.
    ( Red - Densenet, Blue - Resnet no Short, Resnet - yellow, VGG - Brown)
  • The values of loss are scaled to 5% of their original value to allow easier comparison among different models.
  • Each model has different min and max values of loss (Blue to Red) and the color range applies to each model separately.
